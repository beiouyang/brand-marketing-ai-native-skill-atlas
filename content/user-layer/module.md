# 用户层

用户层是整个网站的入口，承接一句话搜索、Hints 推荐和视觉记忆点。

## 页面内容

- 主标题：品牌与营销AI-Native Framwork
- 副标题：把品牌策略、营销生产、互动体验和团队 Skill 沉淀成可复用的 AI-Native 工作系统
- 搜索框：支持 Skill / Agent / Design Wiki / 知识库统一检索
- Hints：提供常用搜索意图入口
- 视觉核心：原生 Canvas 3D 粒子 Torus，支持鼠标和触摸扰动

## Skill 文件夹

`skills/` 用于沉淀与用户入口、检索意图、推荐提示相关的 Skill 或 Agent。

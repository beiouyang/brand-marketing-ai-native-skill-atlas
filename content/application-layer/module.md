# 应用层

应用层用于组织业务场景提效的 Skill / Agent，面向实际业务生产。

## 业务域

- 品牌域：调研策略、IP 资产、视觉一致性
- 营销域：大促会场、资源模块、批量延展
- 互动域：玩法策略、小游戏、Lottie 交付
- 多媒体内容域：动态预演、短剧短视频、内容生成

## 卡片字段

每个 Skill / Agent 至少包含：

- Tag
- Skill / Agent 名称
- Skill / Agent 描述
- 状态：预研/Demo、已应用/版本号、进行中
- 目标和设计价值提效表达

## Skill 文件夹

`skills/` 下按业务域拆分，用于存放真实 Skill 包、说明文档、Prompt、工具脚本或调用清单。
